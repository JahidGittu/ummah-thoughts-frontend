# Quick Reference Guide - Article & Draft Publisher Formatting Tools

## 🚀 Quick Start

### Draft Publisher Page
**Location**: Writer Dashboard → Drafts

**To Format Text**:
1. Type your content
2. Select/highlight the text you want to format
3. Click the formatting button in the toolbar above
4. Formatting applied instantly

**To Add Images**:
1. Click the 🖼 (Image) button
2. Select image from your computer
3. Image appears inline in your content

**To Save**:
1. Click "Save Draft" button
2. Your formatting is preserved

## 📝 Formatting Tools Available

### Text Formatting
| Icon | Name | What It Does |
|------|------|-------------|
| **B** | Bold | Makes text bold: **like this** |
| *I* | Italic | Makes text italic: *like this* |
| <u>U</u> | Underline | Underlines text: _like this_ |

### Headings
| Icon | Name | Result |
|------|------|--------|
| H1 | Heading 1 | # Large heading |
| H2 | Heading 2 | ## Medium heading |
| ¶ | Paragraph | Regular text |

### Lists
| Icon | Name | Result |
|------|------|--------|
| • | Bullet List | • Item 1<br>• Item 2 |
| 1. | Numbered List | 1. Item one<br>2. Item two |

### Other Formatting
| Icon | Name | What It Does |
|------|------|-------------|
| 🔗 | Link | Adds hyperlink to text |
| ≡ | Quote/Blockquote | Creates indented quote |
| — | Divider | Adds horizontal line |

### Alignment
| Icon | Name | Result |
|------|------|--------|
| ⬅ | Align Left | Text starts from left |
| ⬆ | Align Center | Text centered |
| ➡ | Align Right | Text ends at right |

## 🖼 Image Upload Guide

### Draft Publisher
1. Click the **Image** (🖼) button in toolbar
2. Select image from computer
3. Image appears in your content
4. Automatically resized and styled

### Article Publisher
1. Add an "Image" content block
2. Click the upload area
3. Select image file
4. (Optional) Add caption below image
5. Click ✓ (trash icon) to remove

## 📝 Article Publisher Special Features

### Content Blocks
**Available Types**:
- Heading
- Paragraph
- Quote
- Bullet List
- Numbered List
- Code Block
- Image
- Divider

### Per-Block Formatting
1. Click a content block to select it
2. Formatting toolbar appears above the block
3. Tools available depend on block type:
   - **Headings**: Bold, Italic, Underline
   - **Paragraphs**: Bold, Italic, Underline, Strikethrough, Link
   - **Quotes**: Bold, Italic
   - **Lists**: Toggle list type, Bold, Italic
   - **Code**: Language selector
   - **Images**: Caption input

### Code Blocks
1. Add "Code Block"
2. Paste or type code
3. Select language from dropdown:
   - JavaScript
   - Python
   - HTML
   - CSS
   - Bash
   - SQL
   - Java

## 💡 Tips & Tricks

### Draft Publisher

### Article Publisher
1. **Edit Existing**: Click pencil icon on article card to edit

## ❓ Troubleshooting

### Formatting Not Appearing?
✅ **Solution**:
1. Make sure text is selected (highlighted)
2. Click the format button while text is selected
3. Formatting applies to selected text

### Image Not Uploading?
✅ **Solution**:
1. Check file is an image (JPG, PNG, GIF, WebP)
2. File size should be less than 10MB
3. Try again or refresh page

### Lost My Draft?
✅ **Solution**:
1. Drafts are saved in browser
2. Make sure you clicked "Save Draft"
3. Refresh page to reload draft
4. Check other draft cards in list

### Content Lost?
✅ **Solution**:
1. Content with formatting requires saving
2. Click "Save Draft" before closing editor
3. Your formatted content is preserved in saved version

## 🎯 Common Workflows

### Writing a Blog Post (Draft Publisher)
1. Click "New Draft"
2. Enter title
3. Write introduction
4. **Select intro text** → Click Bold for emphasis
5. Add sections with H1 headings
6. Format important terms with **bold**
7. Add images with 🖼 button
8. Click "Save Draft"
9. Later click "Continue Writing" to resume

### Publishing an Article (Article Publisher)
1. Click "New Article"
2. Fill in title and description
3. Add content blocks:
   - Heading for main title
   - Paragraphs for content
   - Images for visuals
   - Lists for organization
4. Format each block as needed
5. Click "Save Draft" to save progress
6. When ready, click "Publish"

### Formatting a Complex Document
1. Use headings to structure
2. Use lists to organize information
3. Use blockquotes for important quotes
4. Use bold/italic for emphasis
5. Use images for visual interest
6. Use horizontal rules between sections
7. Full formatting control over every element
